# node-url-shortner

Follow these steps in order.

1. Fork the repository
2. Run the command
3. Rename config.json.example and change the value of "u" to your mongo URI.
4. Visit `app.js` and change
```js
this.MURI = "..."
```
to your web URL.

5. Run npm i
6. Run npm run prod
7. Open `localhost:9932` and you should be set to go!
